# Breast Cancer Prediction Model

## Overview
-Using the Wisconsin Breast Cancer Dataset, this project aims to compare two classification modesls (decision tree model vs. logistic regression model) in accurately predicting whether a tumor is benign or malignant.
-Performed correlation analysis to identify the strongest predictors (concave points, radius, perimeter) for a breast cancer image.
-Performed a train-test split (80% training, 20% testing) to check how each model performed on the data.

## Results
- Logistic Regression accuracy: 95.6%
- Decision Tree accuracy: 93.9%

## Dataset
Wisconsin Breast Cancer Dataset (built into sklearn)
